## Description

Teste técnico para vaga de Desenvolvedor Backend realizado pela empresa Vila.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# este comando cria os schemas necessários
$ npm run migrate

# este comando popula o banco de dados com o sample
$ npm run seed

# este comando inicia a aplicaçao em watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
